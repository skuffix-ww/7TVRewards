Не работает 7TV?

▼
Токен 7TV истёк

Зайдите на 7tv.app → F12 → Network → обновите страницу → скопируйте заголовок Authorization из запроса к 7tv.io (без «Bearer »).

7TV недоступен в регионе

Включите VPN или прокси и смените регион. Если сайт открывается, но API не отвечает — проблема скорее всего в блокировке на уровне провайдера.

Набор эмоутов не загружается

Выберите набор из выпадающего списка — если 7TV привязан к Twitch-аккаунту, он подтянется автоматически. Или введите ID набора вручную (скопируйте из URL на 7tv.app).

Эмоут не добавляется (conflicting name)

Эмоут с таким именем уже существует в наборе. Система автоматически добавит суффикс (_2, _3...). Если ошибка повторяется — удалите старый эмоут на 7tv.app вручную.

Награда не срабатывает

Выберите награду из списка — это запускает слушатель (polling раз в 5 сек). Убедитесь что авторизация Twitch активна и зритель ввёл корректную ссылку на 7tv.app/emotes/...

Сайт ведёт себя странно / сбросить всё

F12 → Application → Local Storage → удалите ключ 7tv-rewards-state. После обновления страницы всё сбросится до начального состояния.
