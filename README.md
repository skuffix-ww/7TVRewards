# 7TV Rewards

Twitch Channel Points -> 7TV Emotes. Зрители покупают эмоуты за баллы канала, предыдущий эмоут автоматически заменяется новым.

## Возможности

- Авторизация через Twitch OAuth
- Поиск эмоутов через 7TV GQL API
- Выбор набора эмоутов из списка (без ручного ввода ID)
- Добавление/удаление эмоутов через 7TV GQL с Bearer-токеном
- Создание наград канала или выбор существующей
- Настраиваемое описание награды
- Автоматическая замена эмоута при покупке
- Уведомления в чат Twitch (успех/ошибка)
- Пагинация (кнопка "Загрузить ещё") для больших наборов и поиска
- Polling redemptions каждые 5 секунд

## Быстрый старт

### 1. Регистрация Twitch приложения

1. [Twitch Console](https://dev.twitch.tv/console) -> Register Your Application
2. OAuth Redirect URL: `http://localhost:3000`
3. Category: Website Integration
4. Скопируйте **Client ID** и **Client Secret**

### 2. Настройка

```bash
npm install
```

Создайте файл `.env`:

```env
TWITCH_CLIENT_ID=ваш_client_id
TWITCH_CLIENT_SECRET=ваш_client_secret
TWITCH_REDIRECT_URI=http://localhost:3000
PORT=3000
```

В `app.js` укажите тот же Client ID:

```javascript
const CONFIG = {
    TWITCH_CLIENT_ID: 'ваш_client_id',
    TWITCH_REDIRECT_URI: 'http://localhost:3000',
    // ...
};
```

### 3. Запуск

```bash
npm start
```

Откройте `http://localhost:3000`

## Использование

### 1. Авторизация Twitch

Нажмите "Войти через Twitch" и разрешите доступ.

### 2. Настройка 7TV

**Токен 7TV** (нужен для добавления/удаления эмоутов):

1. Откройте [7tv.app](https://7tv.app) и залогиньтесь
2. F12 -> Network -> обновите страницу
3. Найдите любой запрос к `7tv.io`
4. Скопируйте значение заголовка `Authorization` (без `Bearer `)
5. Вставьте в поле "Токен 7TV" и нажмите "Сохранить"

**Набор эмоутов** — выберите из выпадающего списка (загружается автоматически из вашего 7TV профиля) или введите ID вручную.

### 3. Награда

- Создайте новую награду (название, стоимость, описание) или выберите существующую из списка
- Нажмите на награду в списке чтобы начать отслеживание
- Включите/выключите уведомления в чат

### 4. Как работает для зрителей

1. Зритель активирует награду и вводит ссылку на 7TV эмоут
2. Система извлекает ID эмоута из ссылки
3. Проверяет существование эмоута
4. Удаляет предыдущий эмоут из набора
5. Добавляет новый эмоут
6. Отправляет сообщение в чат с результатом
7. Отмечает награду как выполненную (или отменённую при ошибке)

Поддерживаемые форматы:
- `https://7tv.app/emotes/ID`
- `https://7tv.io/emotes/ID`
- Просто ID

## API Endpoints

### 7TV

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/7tv/gql` | GQL прокси (поиск, добавление, удаление эмоутов) |
| GET | `/api/7tv/emote-sets/:setId` | Получение набора эмоутов |
| GET | `/api/7tv/users/twitch/:twitchId` | 7TV профиль по Twitch ID |

### Twitch

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/twitch/users` | Информация о пользователе |
| POST | `/api/twitch/rewards` | Создать награду |
| GET | `/api/twitch/rewards` | Список наград |
| PATCH | `/api/twitch/rewards/redemptions` | Обновить статус активации |
| GET | `/api/twitch/rewards/redemptions` | Невыполненные активации |
| POST | `/api/twitch/chat/messages` | Отправить сообщение в чат |

## Twitch Scopes

- `channel:manage:redemptions` — управление наградами
- `user:read:email` — чтение профиля
- `user:write:chat` — отправка сообщений в чат

## Решение проблем

**Ошибка авторизации** — проверьте Client ID и Redirect URI (должен быть `http://localhost:3000`).

**Не загружаются наборы** — убедитесь что у вас есть 7TV аккаунт, привязанный к Twitch.

**Не добавляются эмоуты** — проверьте 7TV токен. Токен должен быть от аккаунта с правами на редактирование набора.

**DUPLICATE_REWARD** — награда с таким названием уже существует. Выберите её из списка существующих наград.
