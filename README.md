## Как это работает

1. Зритель активирует награду и вставляет ссылку на эмоут с 7tv.app
2. Сервис парсит ID эмоута, проверяет его существование
3. Если слоты заполнены — удаляет самый старый эмоут
4. Добавляет новый эмоут в набор
5. Бот пишет в чат результат (опционально)

Всё это за секунды, пока стример стримит.

## Быстрый старт

### 1. Создать Twitch-приложение

1. Зайти на [Twitch Developer Console](https://dev.twitch.tv/console)
2. Register Your Application
3. OAuth Redirect URL: `http://localhost:3000` (для локального запуска)
4. Category: Website Integration
5. Скопировать **Client ID** и **Client Secret**

### 2. Установка

```bash
git clone https://github.com/skuffix-ww/7TVRewards
cd 7TVRewards
npm install
```

### 3. Конфиг `.env`

```env
TWITCH_CLIENT_ID=your_client_id
TWITCH_CLIENT_SECRET=your_client_secret
TWITCH_REDIRECT_URI=http://localhost:3000
PORT=3000
SEVENTV_API_BASE=https://7tv.io/v3
```

### 4. Запуск

```bash
npm start
# или для разработки
npm run dev
```

Открыть `http://localhost:3000`

## Настройка

### 7TV Токен

Нужен для добавления/удаления эмоутов от вашего имени.

1. Открыть [7tv.app](https://7tv.app) и залогиниться
2. F12 → Network → обновить страницу
3. Найти любой запрос к `7tv.io`
4. Скопировать заголовок `Authorization` (без `Bearer `)
5. Вставить в поле токена в приложении

> Токен периодически истекает — если эмоуты перестали добавляться, обновите токен.

### Набор эмоутов

Приложение автоматически подтягивает ваши наборы если 7TV привязан к Twitch-аккаунту. Либо введите ID набора вручную — скопируйте из URL на 7tv.app.

### Награда канала

Создайте новую награду прямо в приложении или выберите существующую из списка. При выборе существующей — сразу запускается слушатель.

### Слоты эмоутов

Настройте сколько эмоутов хранить до того как начнут удаляться самые старые. Или включите режим «Без ограничений» — тогда набор просто накапливается.

## Возможности

| Фича | Описание |
|------|----------|
| Авто-добавление | Эмоут добавляется сразу после активации награды |
| Слоты | Настраиваемая очередь — старые эмоуты вытесняются новыми |
| Дубли | Проверка перед добавлением, повтор → отмена |
| Conflicting name | Автоматически добавляет суффикс (`_2`, `_3`...) |
| Сообщение в чат | Бот пишет результат — успех, ошибка, дубль |
| Уведомления | Toast-уведомления в интерфейсе (можно отключить) |
| Ручное удаление | Наведите на эмоут → кнопка ✕ |
| Поиск эмоутов | Встроенный поиск по 7TV базе |
| Пагинация | Везде — в наборе, поиске, списке наград |
| Сортировка | Эмоуты набора по алфавиту или по умолчанию |
| Лог событий | Подробный лог всех действий с фильтром |
| Счётчик сессии | Сколько эмоутов добавлено за текущую сессию |
| Летающие эмоуты | Фоновые эмоуты из вашего набора 👻 |
| Пасхалка | Кликните на логотип |

### Advanced Mode

Настройка текста бот-сообщений в чат. Переменные: `{user}`, `{emote}`.

- Сообщение при успехе
- Сообщение при ошибке
- Сообщение при дубликате

## Beta v2.0.0

Активируется через иконку архива в модалке «О приложении».

**Что добавляет:**

- **Дашборд** — история всех добавленных эмоутов
- **Статистика** — баллов потрачено, эмоутов добавлено, уникальных зрителей
- **Лидерборд** — топ зрителей по потраченным баллам
- **Модерация** — бан/мут/блок прямо из истории
  - *Бан* — блокирует добавление эмоутов на выбранное время
  - *Мут* — мутит в Twitch-чате (нужен scope `channel:moderate`)
  - *Блок* — постоянный запрет, все будущие активации отменяются

> Для мута чата нужно перерегистрироваться с дополнительным разрешением. Кнопка «Активировать и перерегистрироваться» сделает это автоматически.

**Планировалось в полном релизе 2.0.0 (не вышел):**
- WebSocket (Twitch EventSub) — без поллинга
- Очередь эмоутов — обработка нескольких активаций одновременно
- OBS overlay с анимацией
- Экспорт/импорт настроек
- Светлая тема
- Cooldown зрителя
- Блэклист эмоутов

## FAQ

**Эмоуты не добавляются**
→ Проверьте 7TV токен — он мог истечь. Обновите по инструкции выше.

**7TV API недоступен**
→ Включите VPN. В некоторых регионах 7tv.io заблокирован на уровне провайдера.

**DUPLICATE_REWARD при создании награды**
→ Награда с таким названием уже существует. Выберите её из списка ниже формы.

**Награда не срабатывает**
→ Убедитесь что награда выбрана в списке — это запускает polling каждые 5 секунд. Зритель должен вставить корректную ссылку вида `https://7tv.app/emotes/...`

**Сбросить всё**
→ F12 → Application → Local Storage → удалить ключ `7tv-rewards-state`
